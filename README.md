# Hangman Game

## CodeAlpha Internship Task

This project is a Python-based implementation of the classic Hangman Game developed as part of the CodeAlpha Internship Program.

## Project Description

The Hangman Game is a command-line application where the player attempts to guess a hidden word one letter at a time. The player has a limited number of attempts to guess the word correctly. The project demonstrates fundamental Python programming concepts such as loops, conditional statements, strings, lists, functions, and user input handling.

## Features

* Random word selection
* Letter-by-letter guessing
* Limited number of attempts
* Win and Lose conditions
* Interactive command-line gameplay

## Technologies Used

* Python

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run the following command:

python hangman.py
